## 🆘 Troubleshooting Common Issues

### The program does not start
Ensure you downloaded the Windows version. If you have an antivirus program, it may flag the file temporarily. Check your antivirus history to see if it quarantined the program. You may need to add an exception for the file if you trust the source.

### The program shows an error about network connections
Check your internet connection. Bilibili may be busy or your firewall may block the connection. Ensure that your browser can open the Bilibili website correctly.

### The program closes immediately after it opens
This often happens if the computer lacks permissions to create files in the current folder. Try moving the application to your Desktop or your Documents folder and run it again.

### The comment file is empty
Verify that the Bilibili video URL is correct. If a video has no comments, the program will create an empty file. Check the browser to ensure the video actually contains comments.
